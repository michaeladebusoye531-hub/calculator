
README – Simple Calculator Project

Project Title

Simple Calculator

Description

This project is a simple calculator developed using HTML, CSS, and JavaScript. It performs basic arithmetic operations including addition, subtraction, multiplication, and division. The calculator features a clean and user-friendly interface that allows users to enter numbers and perform calculations easily.

Features

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Clear (C) button
- Delete (DEL) button (if implemented)
- Responsive and easy-to-use interface

Technologies Used

- HTML
- CSS
- JavaScript

How to Run the Project

1. Download or copy the project files.
2. Ensure all files are in the same folder.
3. Open the index.html file in any modern web browser.
4. Use the calculator by clicking the buttons on the screen.

Project Structure

- index.html – Contains the calculator layout.
- style.css – Handles the design and styling.
- script.js – Contains the calculator logic and functionality.
